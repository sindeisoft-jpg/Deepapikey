# DeepApiKey - 基于DeepSeek网页版的API模拟平台

一款基于Python + PyQt6开发的智能对话应用，通过内嵌浏览器技术直接与DeepSeek网页版进行深度交互，模拟实现类似官方API的功能体验。用户可以通过Qt界面与DeepSeek网页版无缝通信，并支持将对话内容一键导出为Word文档。

## 核心功能

### 🤖 网页版API模拟
- **真实网页交互**：通过JavaScript桥接技术，直接操控DeepSeek网页版
- **API级体验**：模拟官方API调用方式，提供一致的使用体验
- **双向通信**：Qt界面 ↔ 网页DeepSeek ↔ 实时回复展示
- **智能回复捕获**：自动监控网页内容变化，精准获取AI回复

### 📝 文档导出功能
- **一键生成Word**：将对话历史直接导出为格式化的Word文档
- **完整信息保留**：包含时间戳、对话风格、完整问答内容
- **专业格式排版**：自动生成标题、章节和标准文档结构

### 🖥️ 现代化界面
- **双面板设计**：左侧网页浏览 + 右侧对话控制
- **响应式布局**：可自由调整面板比例
- **状态实时反馈**：操作进度和系统状态清晰显示
- **多版本支持**：提供基础版和增强版两种体验

## 技术架构

### 核心原理
本项目通过以下方式实现DeepSeek网页版的API模拟：

1. **网页控制层**：使用PyQt6 WebEngine内嵌DeepSeek网页
2. **JavaScript桥接**：通过runJavaScript()注入脚本控制网页元素
3. **自动化交互**：模拟用户输入、点击发送、监控回复等操作
4. **数据同步**：将网页回复实时同步到Qt界面

### 技术栈
- **Python 3.8+**：稳定的运行环境
- **PyQt6**：现代化GUI框架
- **QtWebEngine**：内嵌浏览器引擎
- **JavaScript注入**：实现网页深度交互
- **pynput**：系统级键盘事件模拟
- **python-docx**：Word文档生成

### 交互流程
```
用户输入 → Qt界面接收 → JavaScript注入网页 → 网页元素操作 → DeepSeek处理 → 回复监控 → Qt界面显示 → 文档导出
```

## 安装部署

### 环境要求
```bash
# 系统要求
- Python 3.8 或更高版本
- 支持的操作系统：Windows/macOS/Linux

# 核心依赖
- PyQt6 >= 6.10.0
- PyQt6-WebEngine >= 6.10.0
- pynput >= 1.7.0
- python-docx >= 0.8.11
```

### 快速安装
```bash
# 1. 克隆项目
git clone <项目地址>
cd appleweb

# 2. 创建虚拟环境
python3 -m venv venv
source venv/bin/activate  # macOS/Linux
# 或 venv\Scripts\activate  # Windows

# 3. 安装依赖
pip install -r requirements.txt

# 4. 启动应用
python main.py
# 或使用增强版
python main_enhanced.py
```

## 使用指南

### 基础操作
1. **启动应用**：运行任一主程序文件
2. **输入问题**：在右侧对话框中输入您的问题
3. **发送消息**：点击发送按钮或按回车键
4. **查看回复**：AI回复将实时显示在对话区域
5. **导出文档**：点击导出按钮生成Word文档

### 高级功能
- **对话风格**：可选择简洁/详细/专业等不同回复风格
- **历史管理**：支持对话历史查看和清空
- **界面调节**：拖拽分割器调整浏览器和对话面板比例
- **状态监控**：实时显示操作状态和系统信息
